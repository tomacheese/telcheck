# AGENTS.md

## 目的
AI エージェント共通の作業方針を定義する。

## 基本方針
- 会話言語: 日本語
- コメント言語: 日本語
- エラーメッセージ言語: 英語
- コミット規約: Conventional Commits
  - `<type>(<scope>): <description>` 形式
  - `<description>` は日本語で記載

## 判断記録のルール
1. 判断内容の要約
2. 検討した代替案
3. 採用しなかった案とその理由
4. 前提条件・仮定・不確実性の明示

## 開発手順（概要）
1. プロジェクトの理解と構造分析
2. `pnpm install` による依存関係のセットアップ
3. 指示に基づく変更の実装
4. `pnpm lint` および `pnpm fix` による品質確認

## セキュリティ / 機密情報
- API キー、認証情報、個人情報を Git にコミットしない。
- ログに機密情報を出力しない。
- 環境変数の適切な利用を徹底する。

## リポジトリ固有
- Yamaha NVR510 との通信を行うプロジェクトであるため、ネットワーク関連の知識を前提とする。
- `data/config.json` が唯一の設定ファイルであり、その構造は `schema/Configuration.json` に定義されている。
